# Frontend CI (정적 산출물 패키징만)

## 목적
- 커밋/태그 시 프런트 정적 산출물을 빌드하고 패키징한다.
- 업로드/스위치는 다음 단계에서 파이프라인에 추가한다.

## 트리거
- 태그: `v` 또는 `v-rc.`
- 브랜치(선택): `develop`에만 실행

## 규칙(배포 경로)
- S3 버전 경로: `app/<git-sha>/...`
- 현재 버전 별칭: `app/latest/`(스위치 전용)
- CloudFront 무효화: HTML만 대상 (정적 자산은 파일명 해시로 해결)

## 단계
1) 체크아웃  
2) 빌드(정적 파일 생성)  
3) 산출물 패키징(후속 S3 업로드용 아카이브 생성)  
4) 산출물 이름에 `<git-sha>` 포함

## 수락 기준
- 빌드/패키징까지만 정상 완료, 업로드/무효화는 아직 수행하지 않음
